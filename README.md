# Logistica 2.0

Logistica is an item transport, distribution and storage mod for [Luanti](https://www.luanti.org).

Logistica 2.0 is a massive upgrade to Logistica featuring many new concepts, logic and useful machines! Check out the in-game guidebook to get started.

The core principle behind this mod is that item transportation is demand-driven: no item is moved unless it is needed somewhere.

# Download
Available for download at ContentDB: https://content.luanti.org/packages/ZenonSeth/logistica/
